# Blogging Site

## Introduction

Simple blogging site to write and store your personal blogs on your computer.

----

## Tech Stack

- Vanilla JS (frontend)
- Node JS (backend)

---

## How to run

Clone the repository and then navigate to the "backend" directory in "Ver-1.0" inside your terminal, and run:

<pre>npm install</pre>

to install all dependencies the repository.

Then, inside the "backend" directory, type:

<pre>node server.js</pre>

Open *localhost:5000* to see the app runnning.
